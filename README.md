## How to run
- `cd philo`
- `make all`
- `./philo [n] [limit] [eat duration] [sleep duration] [eat reps]`
  - n: 철학자 수
  - limit: 식사하지 않고 버틸 수 있는 시간
  - eat duration: 식사에 소요되는 시간
  - sleep duration: 수면에 소요되는 시간
  - eat reps: 식사 횟수, 모든 철학자가 최소 이 횟수 만큼 식사할 경우 시뮬레이션 종료
